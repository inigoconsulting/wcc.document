wcc.document Installation
-------------------------

To install wcc.document using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.document`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.document

* Re-run buildout, e.g. with:

    $ ./bin/buildout
